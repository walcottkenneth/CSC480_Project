Tips to improve PC performance in Windows

  1. unnecessary software to free up disk space. This is done by visiting the programs and features section. From there, you'll find a record of which apps you've used recently, and which take up the most space. After surveying what’s installed, you can decide which programs can be deleted to help improve system performance.
  2. Determine which applications run when your PC boots up. To access Task Manager, press Ctrl-Shift-Esc. A box will appear that lists all the applications you have installed on your computer. It also gives you a detailed account of the amount of RAM each program uses when you start your PC. To make an adjustment, just right-click on the application you want to change to tell it not to run until commanded.
  3. Check for spyware and viruses. To customize how your device is protected with these Windows Security features select Start > Settings > Update & Security > Windows Security and then Virus & threat protection. Under Current threats, select Quick scan. To run an advanced scan in Windows Security, Under Current threats, select Scan options (or in early versions of Windows 10, under Threat history, select Run a new advanced scan). select Full scan (check all files and programs currently on your device). Select Scan now.
  4. Use Disk Cleanup and defragmentation. Open Disk Defragmenter by clicking the Start button. In the search box, type "Disk Defragmenter", and then, in the list of results, click Disk Defragmenter. Under Current status, select the disk you want to defragment. To determine if the disk needs to be defragmented or not, click Analyze disk. Administrator permission required If you're prompted for an administrator password or confirmation, type the password or provide confirmation. Once Windows is finished analyzing the disk, you can check the percentage of fragmentation on the disk in the Last Run column. If the number is above 10%, you should defragment the disk. Click Defragment disk. Administrator permission required If you're prompted for an administrator password or confirmation, type the password or provide confirmation. Disk Defragmenter might take from several minutes to a few hours to finish, depending on the size and degree of fragmentation of your hard disk. You can still use your computer during the defragmentation process.
  5. Delete Web browser Temporary files and website data.

Computer won’t start

  1. Turn off the computer.
  2. Disconnect the AC adapter.
  3. Remove any discs from the disc drive.
  4. Remove memory cards from the card reader slot.
  5. Disconnect all nonessential devices such as printers, scanners, external hard drives, and USB flash drives. NOTE: Do not disconnect your mouse, keyboard, or monitor.
  6. With the AC adapter unplugged, press and hold the power button for approximately 15 seconds.
  7. Reconnect the AC adapter, and then turn on the computer

Mouse & Keyboard Issues

  1. Look for and remove any packaging tabs from the device. Tabs prevent the device from receiving power.
  2. Check the power connection status: If the device has an On/Off switch, make sure it is set to on.
  3. If the device does not turn on, recharge the battery by connecting the charging cable to a power source, such as a USB charging port on the computer or an external USB power adapter. If the device uses replaceable batteries, install new ones.
  4. Locate the USB receiver for the device. It may already be inserted in a USB port on the computer, or in the original packaging. Some HP Touch and All-in-Ones come with the USB receiver preinstalled in the computer. Or look for a storage container for the receiver inside the device.
  5. Make sure the receiver is inserted in a USB port on the computer. A USB hub might not provide enough power for your device to connect properly.
  6. Turn off any other wireless devices that you are not using, including Bluetooth devices.
  7. Move the mouse closer to the wireless receiver, usually within 30 cm (1 ft).
  8. Make sure there are no objects between the mouse and the wireless receiver. Objects such as TVs, cordless phones, or speakers can weaken wireless signals.
  9. Reset the USB receiver connection based on if it has a Connect button or not.
  10. For Bluetooth devices, make sure Bluetooth is on and keyboard is listed under devices. If not, place the device in discovery mode and click and add device.
  11. Make sure hardware drivers are up to date.

Monitor Issues

  1. Check for power by pressing the power button. If the monitors power indicator is off, then the monitor is not receiving power.
  2. Check that the cables are correctly connected. If the power indicator is on, check the cable connection from the monitor to the computer.
  3. Connect the computer to another monitor to determine if the problem lies with the monitor or the computer.
  4. Inspect the video cable and video connectors for damage.

Troubleshooting Wireless Network

  1. Check the wireless signal and make sure you are connected to the network.
  2. Update hardware drivers by opening device manager and right clicking on the wireless adapter to update driver.
  3. Use the Windows Network and Internet Troubleshooter to test and repair the network connection. In the search box type and select "find and fix network problem". Follow the instruction to troubleshoot your network connection.
  4. Reset the Wireless router by disconnecting the power cord for 20 seconds before reconnecting.
  5. Run a Windows update. In the search window type and select "check for updates".

Zoom Audio Issues

  1. Check if the microphone, speakers, or headphones are plugged in and not muted.
  2. Unplug the device, then plug it back in.
  3. If it's a Bluetooth device, ensure that it is turned on and connected properly.
  4. Make sure that you are using a supported USB device for Zoom. Note: USB devices not listed within the above article may still work with Zoom. However, the devices listed are what we recommend for the most optimal experience. Supported USB devices less likely encounter audio issues.
  5. Ensure that no other applications are using the microphone at the same time.
  6. Test and check audio settings. In the zoom app settings, make sure the speaker settings are configured correctly.

Scheduling with Microsoft Outlook plugin for desktop

NOTE: You can download the Zoom Microsoft Outlook Plugin MSI file from the Download Center. Once the download is complete, run the MSI file and follow the install wizard to complete the installation. You will need to restart Outlook before using the Plugin.

  1. Add plugin icon in outlook: Outlook options > add-ins >disabled items > zoom outlook plug-in > select enable > select Com add-ins > check zoom outlook plugin box.
  2. Select schedule a meeting button within outlook.
  3. Select desired settings for the meeting (ID, security, video, audio, participants, etc.)
